## What you will learn

By the end of this tutorial you will be able to:

- Install `aurora-catalyst-cli` as a dev dependency
- Author a minimal `{bc}.aurora.yaml` schema
- Run `catalyst generate` and understand the output
- Recognize the role of the lockfile and preservation regions

## Prerequisites

- Node.js ≥ 18
- An empty Aurora Catalyst project (or follow the scaffold steps below)
