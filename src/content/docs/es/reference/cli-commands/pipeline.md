## `catalyst pipeline`

Generate a deploy workflow from templates (interactive).

```
USAGE
  $ catalyst pipeline [-f]

FLAGS
  -f, --force  Overwrite existing workflow file

DESCRIPTION
  Generate a deploy workflow from templates (interactive).

EXAMPLES
  $ catalyst pipeline

  $ catalyst pipeline --force
```
