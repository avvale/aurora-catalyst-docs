* [`catalyst keys`](#catalyst-keys)

## `catalyst keys`

Generate RSA private key and public key

```
USAGE
  $ catalyst keys [-o <value>]

FLAGS
  -o, --out=<value>  Output directory for the key files

DESCRIPTION
  Generate RSA private key and public key

EXAMPLES
  $ catalyst keys

  $ catalyst keys --out ./keys
```
