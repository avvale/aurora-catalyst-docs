`catalyst help`
===============

Display help for catalyst.

* [`catalyst help [COMMAND]`](#catalyst-help-command)

## `catalyst help [COMMAND]`

Display help for catalyst.

```
USAGE
  $ catalyst help [COMMAND...] [-n]

ARGUMENTS
  [COMMAND...]  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for catalyst.
```

_See code: [@oclif/plugin-help](https://github.com/oclif/plugin-help/blob/6.2.44/src/commands/help.ts)_
