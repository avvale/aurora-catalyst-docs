## Qué vas a aprender

Al terminar este tutorial vas a ser capaz de:

- Instalar `aurora-catalyst-cli` como dev dependency
- Escribir un `{bc}.aurora.yaml` mínimo
- Ejecutar `catalyst generate` y entender el output
- Reconocer el rol del lockfile y las preservation regions

## Requisitos previos

- Node.js ≥ 18
- Un proyecto Aurora Catalyst vacío (o seguí los pasos de scaffold más abajo)
